# AIチャットを体験しよう

## (STEP0) AIチャット体験を始める前に

### (STEP0-1) 自己紹介しましょう

ハンズオンをするグループのメンバーに各自1分間の自己紹介をしましょう。

### (STEP0-2) ハンズオンで使用するウィンドウ操作を確認しよう

- このGitHubサイトが表示されているウィンドウを画面の左半分に表示させよう
  - [Windowsの場合]：ウィンドウをクリックして「Windows」キー＋「←」キーを押下
  - [Macの場合]：ウィンドウの緑●にマウスを合わせて「ウィンドウを画面の左側にタイル表示」を選択
- 新しく開いたウィンドウを画面の右半分に配置しよう
  - [Google](https://www.google.com)を右クリックして「新しいウィンドウで開く」
  - [Windowsの場合]：ウィンドウをクリックして「Windows」キー＋「→」キーを押下
  - [Macの場合]：ウィンドウの緑●にマウスを合わせて「ウィンドウを画面の右側にタイル表示」を選択
- ウィンドウを最大化しよう
  - [Windowsの場合]：ウィンドウの右上の「□」アイコンをクリック
  - [Macの場合]：「Option」キーを押しながらウィンドウの緑●をクリック
- ウィンドウを全画面表示しよう
  - [Windowsの場合]：ウィンドウをクリックして「F11」キーを押下
  - [Macの場合]：ウィンドウの緑●をクリック
- ウィンドウを全画面表示を解除しよう
  - [Windowsの場合]：ウィンドウをクリックして「F11」キーを押下
  - [Macの場合]：ウィンドウの緑●をクリック
- ウィンドウを閉じよう
  - [Windowsの場合]：ウィンドウの右上の「×」アイコンをクリック
  - [Macの場合]：ウィンドウの赤●をクリック

### (STEP1-3) Geminiでスライドを作成してみよう

- プロンプト入力欄を「思考モード」に設定します
  - 「思考モード」が使用制限で使えなくなっている場合は「高速モード」のままでOKです
- 「ツール」をクリックして「Canvas」を選択し、画面に「Canvas×」と表示されるようにします
- 「Geminiへのプロンプトを入力」と書かれたプロンプト入力欄に下記のプロンプト例をコピーして、背景色や文字色を変更しましょう
- さらにその下に「スライド内容」をコピペして紙飛行機アイコン「＞」をクリックします
  - スライド生成まで約1分ほどかかります

**プロンプト例**

```
下記の設定でその下にあるスライドの内容を反映したプレゼンスライドを作成して

タイトルの文字色: 濃いネイビー
本文の文字色: 濃いグレー

```

**スライド内容(1)**

```
タイトル：「シーサーの口の“開け方”に意味があるって知ってますか？」
スライド1：家の門に並ぶ2体のシーサー写真
「沖縄の家には必ずある“シーサー”。実は口の開け方に意味があるんです。」
スライド2：左が口を開け、右が閉じた写真
「左側が“開き口”で、悪いものを追い払う役割。右側は“閉じ口”で、福を逃さない役割を持っています。」
スライド3：歴史・由来の簡単図解
「由来は中国の“獅子像”で、守護と繁栄の象徴。琉球王国時代から家を守る風習として広まりました。」
スライド4：現代的なシーサー（カラフルな土産品）
「今ではカラフルでポップなシーサーも人気。昔と今の文化の融合が見える面白い例です。」
スライド5：まとめ
「今度沖縄に行ったら、シーサーの“口”をぜひチェックしてみてください。」
```

**スライド内容(2)**

```
タイトル：「那覇市の裏通りで見つけた“地元の宝”」
スライド1：那覇市の地図＋シーサーの写真
「私は沖縄県那覇市出身です。観光地のイメージが強いですが、地元ならではの“暮らしの温かさ”が魅力です。」
スライド2：国際通りの夕景
「まずおすすめは“国際通り”。でも観光ではなく、夕方の地元時間。放課後に友達と寄った小さなカフェの雰囲気が忘れられません。」
スライド3：地元の食“ゆし豆腐そば”の写真
「次に紹介したいのが“ゆし豆腐そば”。優しい味で、飲み会の翌朝に必ず食べます。観光客より地元民に愛される味です。」
スライド4：路地裏の看板ネコの写真
「最後は“猫”。那覇の裏道ではよく猫が昼寝していて、観光地の喧騒を忘れられます。ぜひ探してみてください。」
スライド5：まとめ
「観光だけでなく、地元の人の日常を歩くと、那覇の魅力はもっと深まります。沖縄に来たら、ぜひ路地を曲がってみてください。」
```

**スライド内容(3)**

```
タイトル：「疲れた日に食べたい！私の“ベスト3スイーツ”」
スライド1：タイトル＋スイーツ写真3種
「甘いものが大好きな私が選ぶ、“ベスト3スイーツ”を紹介します！」
スライド2：第3位『プリン』
「シンプルで王道。特に硬めの昔ながらのプリンが好みです。小さな喫茶店で見つけるとテンションが上がります。」

スライド4：第1位『紅いもタルト』
「地元沖縄の味。鮮やかな紫色とほくほく感がたまりません。お土産にも喜ばれます。」
```

- スライドが作成されたら右上の「スライドにエクスポート」をクリックしましょう

## (STEP2) NotebookLMを使ってみよう

**◆ハンズオン内容◆** : NotebookLMを使って情報をまとめる方法を体験しよう

### (STEP2-1) NotebookLMにログイン

- [NotebookLM](https://notebooklm.google/)を右クリックして「新しいウィンドウで開く」
- 「Try NotebookLM/NotebookLMを試す」あるいは「ブラウザでNotebookLMを試す」をクリックして、Googleアカウントでログイン
- 下部にCookieの同意が求められたら「同意する」をクリック

※ Googleアカウント登録をしていない方→ [Googleアカウント作成](0_GoogleAccount.md)

### (STEP2-2) NotebookLMに情報ソースを設定しよう

- 「最初のノートブックを作成する」画面になったら「作成」ボタンをクリック
  - すでにノートブックを作成済みの場合は「＋ノートブックを新規作成」をクリック
- 「ソースを追加」ウィンドウが開く
- 下記のリンクのいずれかのリンクを右クリックして「リンクのアドレスをコピー」でURLをコピー
  - Youtube検証動画(6分9秒) : [携帯電話を使いながら充電するとバッテリーが消耗しますか?](https://www.youtube.com/watch?v=Lj4LMlGr4og)
  - Youtube紹介動画(17分56秒) : [Touring Dell's Factory that Builds AI Factories](https://www.youtube.com/watch?v=8BSA67BqWjw)
  - Youtube講演動画(42分33秒) : [Deep dive into AI tools in Copilot Studio | BRK315](https://www.youtube.com/watch?v=BLQ8XZ5PXF4)
  - Pew Research Center調査レポート : [How People Around the World View AI](https://www.pewresearch.org/global/2025/10/15/how-people-around-the-world-view-ai/)
- 「～から音声解説と動画解説を作成する」ウィンドウの下部にある「または、ここにファイルをドラッグ＆ドロップする」の下にある「ウェブサイト」をクリックして、「リンクを貼り付ける」欄にコピーしたURLを貼り付けて「挿入」をクリック
- 左ペインに表示されたソースの名前の右にチェックマークがついたら読み込み完了
- 中央ペイン下部あるいはチャットタブにある「入力を開始します…」と書かれたプロンプト欄に下記のテキストをコピーして貼り付け後、「→」アイコンをクリック

```
ソースの内容を要約して
```

### (STEP2-3) NotebookLMでソースの情報をまとめてもらおう

- インフォグラフィックの作成 (無料ユーザーは1日3つ作成可能)
  - Studioとかかれた右ペインの「スライド作成」欄の右にあるペンアイコンをクリック
  - 形式を「プレゼンターのスライド」、言語を「日本語」、長さを「短め」に設定して「生成」をクリック
  - 「インフォグラフィックを生成しています…」と表示され矢印が回転しはじめる
  - 右ペイン下部に棒グラフアイコンでインフォグラフィックが作成される
  - クリックすると生成されたインフォグラフィックを確認できる。右上の×アイコンクリックで閉じることが可能
- スライドの作成 (無料ユーザーは1日1つ作成可能)
  - Studioとかかれた右ペインの「スライド作成」欄の右にあるペンアイコンをクリック
  - 形式を「プレゼンターのスライド」、言語を「日本語」、長さを「短め」に設定して「生成」をクリック
  - 「スライド資料をを生成しています…」と表示され矢印が回転しはじめる
  - 右ペイン下部に四角いアイコンでスライド資料が作成される
  - クリックすると生成されたスライド資料を確認できる。右上の」「を上下逆にした形のアイコンクリックで閉じることが可能。スライド名称の右の「」を上下逆にした形のアイコンクリックで拡大表示できる。拡大表示は右上の×アイコンクリックで閉じることが可能

## (STEP4) ディープリサーチを体験しよう

**◆ハンズオン内容◆** : 推論モードでAI検索を実施するディープリサーチを体験しよう

### (STEP4-1) Feloを使ってディープリサーチを体験しよう

- [Felo](https://felo.ai/)を右クリックして「新しいウィンドウで開く」
- 右上の「登録」アイコンをクリック
- 「Feloへようこそ」ウィンドウになるので「Googleで続行」をクリックしてGoogleアカウントでログイン
- プロンプト欄の右下のProをチェックした後、モデルを「Best」に変更
- プロンプト欄上部の「検索」を「リサーチ分析」に変更
- 「研究テーマを入力…」と書かれたプロンプト欄に下記の文を入力して「→」をクリックしよう

```
日本政府のAI戦略について方針と具体的な施策について調べてまとめてください
```
