# プレゼンテーションの作成

## スライド内容の検討

### スライド内容のメモ用アプリ起動

- メモ帳などの「3分プレゼンハンズオン(基本構成とおすすめテーマ)」のスライド内容をメモするアプリを起動しましょう。
- 下記に「Googleドキュメント」をメモ用アプリとして起動する手順を紹介します。

#### Googleドキュメントの作成

- Googleドライブへのアクセス
  - [Google](https://www.google.com)を右クリックして「新しいウィンドウで開く」
  - ログインしていない場合は、右上のログインをクリックしてGoogleアカウントでログイン
  - アカウントアイコンのすぐ左にある「Googleアプリ」のアイコンをクリック
  - 「ドライブ」アイコンをクリックして「Googleドライブ」にアクセスします
- Googleドキュメントの作成
  - Googleドライブの左のメニューの「マイドライブ」をクリック
  - Googleドライブの左上の「+新規」をクリック
  - 「Googleドキュメント」をクリック
  - 「無題のドキュメント」と書かれている部分がファイル名なので、「スライドメモ」などのファイル名に変更してください

※ Googleアカウント登録をしていない方→ [Googleアカウント作成](../Preparation/GoogleAccount.md)

### メモ用アプリで発表スライド内容を作成

- [3分プレゼンテーションの基本構成とおすすめテーマ](1_Theme.md)のおすすめテーマの中から選んだテーマのサンプルプレゼンテーションのパートのテキストをコピーして、メモ用アプリへ貼り付けてください
- 貼り付けた内容を修正して、発表するスライド内容を作成しましょう
  - タイトルを修正しましょう
  - スライドXの：の箇所にはスライドに貼る画像イメージを記述、あるいはスライドタイトルを記述しましょう
  - スライドXの下の箇条書き部分にはスライドに記述する文章を記述しましょう

### AIチャットのGeminiにスライド内容作成のサポートをしてもらう

#### Geminiへのログイン

- [Gemini](https://gemini.google.com/)を右クリックして「新しいウィンドウで開く」
- 右上の「ログイン」アイコンをクリックしてGoogleアカウントでログイン

#### Geminiへ質問

- 「Geminiに相談」と書かれたプロンプト入力欄の右下が「高速モード」あるいは「思考モード」になっているので、ここを「高速モード」に変更します
- 「Geminiに相談」と書かれたプロンプト入力欄に質問内容を記入して紙飛行機アイコン「＞」をクリックします

質問サンプル

```
三重県のおすすめスポットを10箇所挙げて、おすすめ理由を教えてください
```

```
京都府のおすすめグルメを10個挙げて、その特徴を教えてください
```

```
「食べるの大好き、調味料に凝っている、体重が気になる」という3つの特徴から、自己紹介のスライド資料に使えるように、
これらの特徴をポジティブにまとめた短いキャッチフレーズ（一文）を作成してください。
```

```
ゆいレールのうんちくを三つの文で紹介したいので、内容を作成してください
```

## Geminiを利用してスライド内容からスライドを生成

### Geminiへのログイン

- Geminiにスライド内容作成サポートをしてもらっていない場合はGeminiにログインしてください
  - [Gemini](https://gemini.google.com/)を右クリックして「新しいウィンドウで開く」
  - 右上の「ログイン」アイコンをクリックしてGoogleアカウントでログイン

### Geminiを用いたスライド作成

- 「Geminiに相談」と書かれたプロンプト入力欄の右下が「高速モード」あるいは「思考モード」になっているので、ここを「思考モード」に変更します
  - 「思考モード」が使用制限で使えなくなっている場合は「高速モード」のままにしておきましょう
- 「ツール」をクリックして「Canvas」を選択し、画面に「Canvas×」と表示されるようにします
- 「一緒に文章を書いたり、コンテンツを作成したりしましょう」と書かれたプロンプト入力欄に下記のプロンプト例をコピーして、背景色や文字色を変更しましょう
- さらにその下にメモ用アプリで作成した発表スライド内容をコピペして紙飛行機アイコン「＞」をクリックします
  - スライド生成まで約1分ほどかかります

```
下記の設定でその下にあるスライドの内容を反映したプレゼンスライドを作成して
背景色: 白
タイトルの文字色: 濃いネイビー
本文の文字色: 濃いグレー

```

発表スライド内容(サンプル)

```
タイトル：「シーサーの口の“開け方”に意味があるって知ってますか？」
スライド1：家の門に並ぶ2体のシーサー写真
「沖縄の家には必ずある“シーサー”。実は口の開け方に意味があるんです。」
スライド2：左が口を開け、右が閉じた写真
「左側が“開き口”で、悪いものを追い払う役割。右側は“閉じ口”で、福を逃さない役割を持っています。」
スライド3：歴史・由来の簡単図解
「由来は中国の“獅子像”で、守護と繁栄の象徴。琉球王国時代から家を守る風習として広まりました。」
スライド4：現代的なシーサー（カラフルな土産品）
「今ではカラフルでポップなシーサーも人気。昔と今の文化の融合が見える面白い例です。」
スライド5：まとめ
「今度沖縄に行ったら、シーサーの“口”をぜひチェックしてみてください。」
```

- スライドが作成されたら右上の「スライドにエクスポート」をクリックしましょう
  - スライドエクスポートに約1分ほどかかります
- 「新しいスライドが作成しました。スライドを開く」というウィンドウが左下に開くので、「スライドを開く」をクリックすると新しいタブで作成されたGoogleスライドが開きます

## Googleスライドの編集

### Googleスライドの基本操作

- 新規スライドの作成：虫眼鏡アイコンの右の「+」(新しいスライド)をクリック
- 文字列の入力エリア作成：矢印アイコンの右の「[T]」(テキストボックス)アイコンをクリック
- 画像の挿入：矢印アイコンの二つ右の「山のイラスト」(画像の挿入)アイコンをクリック
  - 「パソコンからアップロード」を選択後、ファイルを選択
- 図形の入力：矢印アイコンの三つ右の「○□」(図形)アイコンをクリック
  - 図形＞長方形と選択して配置
- 線の入力：矢印アイコンの四つ右の「線」(線)アイコンをクリック
- スライドの保存：自動的に保存されるので保存する操作は不要
- スライドショーの開始：右上の「スライドショー」ボタンをクリック

### Googleスライド/PowerPointのお薦めキーボードショートカット

- 表示
  - Ctrl+「+」:表示の拡大
  - Ctrl+「-」:表示の縮小
- 文字
  - Ctrl+「b」：太字のオン・オフ
  - Ctrl+Shift+「>」：文字を拡大
  - Ctrl+Shift+「<」：文字を縮小
- オブジェクト
  - Ctrl + マウスで移動：選択したオブジェクトをコピー
  - Shift + マウスで移動：垂直方向、水平方向の向きに合わせる
  - Ctrl + Shift + マウスで移動：垂直方向、水平方向の向きに合わせてコピー
  - Shift + マウスでサイズ変更：オブジェクトの縦横比を保ったままサイズ変更

## 画像素材の差し替え

### 画像素材サイト

- [AdobeStock](https://stock.adobe.com/jp/)
  - 「無料素材」にして検索すると無料の画像素材が検索できます
  - ダウンロードするにはGoogleアカウントでのログインが必要です
  - 汎用的な画像素材はここで入手可能です
- [PhotoAC](https://www.photo-ac.com/)
  - ダウンロードするにはGoogleアカウントでのログインが必要です
  - 日本の「街並み・文化・食べ物」の素材が豊富
  - 無料ユーザーが画像検索できる一日4回までという制限があります
- [ぱくたそ](https://www.pakutaso.com/)
  - ダウンロードするのにログインは不要です
  - 困った、嬉しい、悲しいなどのキーワードで日本人画像を検索可能です
- [Pixabay](https://pixabay.com/ja/)
  - ダウンロードするのにログインは不要です
- [Unsplash](https://unsplash.com/ja)
  - ダウンロードするのにログインは不要です

### 画像生成AIで画像生成

- 画像生成ツールWhiskへアクセス
  - [Whisk](https://labs.google/fx/tools/whisk)を右クリックして「新しいウィンドウで開く」
  - アカウント選択画面になるのでGoogleアカウントでログインします
  - 中央にある「ツールを開く→」をクリックしてください
  - 「Whiskへようこそ」ウィンドウが開くので右上の×をクリックして閉じます
- 画像の生成
  - 「アイデアについて説明するか、サイコロを振ってアイデアを広げます」と書かれたプロンプト入力欄に画像生成プロンプトを入力して「→」をクリックしてください
  - 生成された画像の右上の「↓」アイコンをクリックすれば画像をダウンロードできます
- 画像の編集・生成履歴
  - 「編集する」をクリックすると画像の編集モードになります。「編集内容について説明してください」の欄に「線画に変更して」などの指示を与えると、その指示に従った修正が実施されます
  - 編集モードは右上の「調整モード×」のアイコンをクリックすると終了します
  - 右上の「マイライブラリ」をクリックすると新しいタブで画像の生成履歴が閲覧可能になります
